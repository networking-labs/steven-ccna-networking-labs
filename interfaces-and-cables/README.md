# Interfaces and Cables

## Objective
Learn about different network interfaces and cabling standards.

## Key Concepts

### RJ-45
- Standard connector for Ethernet cables (twisted pair)
- Commonly used for LAN connections
- Used on the end of a copper Ethernet cable

### Ethernet
- Network technology used to connect devices in LANs
- Works with RJ-45 and twisted-pair cables

### Network Protocols
- Rules governing how devices communicate
- Examples: TCP/IP, UDP
- Physical standards like connectors and cables, as well as Logical standards like TCP/IP

### Bits and Bytes
- Basic units of digital data
- Understanding how data is transmitted over networks

### Ethernet Standards
- 10BASE-T → 10 Mbps → 100m max length → 4 wires 
- 100BASE-T → 100 Mbps → 100m max length → 4 wires 
- 1000BASE-T → 1 Gbps → 100m max length → 8 wires 
- 10GBASE-T → 10 Gbps → 100m max length → 8 wires 

### UTP Cables
- Unshielded Twisted Pair cables
- Variants: Cat5e, Cat6, Cat6a, Cat7

### Fiber-Optic Connections
- High-speed communication over glass fibers
- Multimode Fiber → short distances
- Single-Mode Fiber → long distances

### Cable Types
- Straight-through → connect PC to switch
- Crossover → connect similar devices (PC-PC, Switch-Switch)
- Auto MDI-X → automatically detects cable type
